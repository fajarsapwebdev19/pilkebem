Aplikasi PILKEBEM V 3.0

Fitur Aplikasi :
=> Support php 7 >= (Not Recomended php 8 >=)
=> Proses Ubah Pengaturan Menggunakan Ajax
=> Export Dan Import Data Mahasiswa Dan Dosen
=> Generate Password Mahasiswa Dan Dosen
=> Cetak Data Menggunakan Dompdf
=> Tampilan Responsive
=> Pemilihan Hanya Berlaku Sekali
=> Batas Pengiriman Vote
=> Buku Panduan Admin Dan User

Cara Install

Manual
=> Download Xampp versi 7.1 atau selebihnya recomended 7.4
=> Download Source Codenya
=> Extract taruh di dir (C:/D:/E:) xampp/htdocs
=> buka localhost/phpmyadmin di browser
=> create database kasih nama bebas (contoh: e_pilkebem)
=> import database dir (C:/D:/E:) xampp/htdocs/(nama_file_anda)/database/e-pilkebem
=> aplikasi siap di gunakan

Mengunakan git bash
=> buka halaman directori htdocs anda
=> klik kanan kemudian pilih gitbash
=> git clone https://github.com/fajarsapwebdev19/pilkebem.git
=> tunggu proses downloadnya selesai
